RSA Multi-User Demo

Create multiple users, generate RSA key pairs, encrypt messages for selected recipients, and decrypt with user private keys.

Encrypt

How it works

  • Add users and generate RSA key pairs for each.
  • Select a recipient in the Encrypt tab to auto-fill their public key.
  • Encrypt plaintext to Base64 ciphertext using the recipient's public key.
  • Select the user in the Decrypt tab to decrypt with their private key.
